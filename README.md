Definition Softwareprodukt
Produktname: Spoolsmith
Slogan: Manage your 3D prints smarter, from spool to sale.
Kurzbeschreibung: SpoolSmith ist eine Webanwendung zur Verwaltung von 3D-Druck-Projekten, Filamentbeständen, Druckkosten und kleinen Kundenaufträgen. Die Software richtet sich an private Maker und kleine 3D-Druck-Verkäufer.
Abgrenzung: SpoolSmith ist kein Slicer und keine Druckersteuerung. Die Anwendung ersetzt also keine Programme wie Bambu Studio, Cura oder PrusaSlicer, sondern ergänzt diese durch Projektverwaltung, Kostenkalkulation und Auftragsorganisation.

Funktionen
Hauptfunktionen:

1. Projektverwaltung
Nutzer können Druckprojekte mit Name, Beschreibung, Bild, Material, Druckdauer, Druckprofil und Notizen speichern.

2. Filamentlager
Filamentrollen können mit Material, Farbe, Hersteller, Restmenge, Preis und Lagerort verwaltet werden.

3. Kostenrechner
SpoolSmith berechnet geschätzte Kosten für Material, Strom, Maschinenzeit, Verschleiß und Fehldruckquote.

4. Auftragsverwaltung
Kleine Verkäufer können Kundenaufträge mit Status wie geplant, gedruckt, nachbearbeitet, geliefert oder bezahlt verwalten.

5. Lizenznotizen
Für STL-Dateien und 3D-Modelle kann dokumentiert werden, ob private oder kommerzielle Nutzung erlaubt ist.

6. Export
Projekte, Kostenaufstellungen und Aufträge können als PDF oder CSV exportiert werden.
